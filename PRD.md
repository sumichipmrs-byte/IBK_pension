# PRD: IBK 변액연금보험 판매용 랜딩페이지 제작

## 1. 프로젝트 개요
- **목적**: IBK 변액연금보험 상품의 장점을 효과적으로 전달하여 가망 고객의 상담 신청(DB 확보)을 유도함.
- **핵심 상품**: IBK 변액연금보험 (8% 평생연금 보증형)
- **주요 타겟**: 노후 준비를 고민하는 30~50대 직장인 및 자영업자.

## 2. 핵심 가치 제안 (Key Value Propositions)
1. **8% 최저보증**: 수익률과 상관없이 납입 원금의 단리 8%를 연금 기준 금액으로 최저 보증.
2. **안정성**: 변액 상품임에도 불구하고 원금 손실 걱정 없는 노후 준비 가능.
3. **비과세 혜택**: 관련 세법 충족 시 연금 수령액 전체 비과세.
4. **상속 가능**: 조기 사망 시에도 '최저 연금 기준액'을 가족에게 보장.

## 3. 랜딩페이지 주요 섹션 구성 (Structure)
1. **Hero Section (상단)**: 
   - 강렬한 헤드라인: "수익률 걱정 NO! 원금의 8% 평생 보증하는 IBK 변액연금"
   - 시각적 요소: 평온한 노후 생활 이미지, 8% 강조 그래픽.
   - CTA: 무료 상담 신청하기 버튼.
2. **Problem/Solution Section**: 
   - 기존 변액보험의 낮은 수익률과 원금 손실 우려를 언급.
   - IBK 8% 최저보증 상품을 대안으로 제시.
3. **Simulation Section (시뮬레이션)**:
   - 40세 남성, 월 50만원 납입 시 예시.
   - 65세부터 매달 약 67.6만원 수령 가능.
   - 80세 생존 시 원금의 2배, 100세 생존 시 원금의 4배 이상 수령 가능성 제시.
4. **Safety Net Section (사후 보장)**:
   - 사망 시에도 남은 가족에게 지급되는 '최저 연금 기준액(예: 1억 4,720만원)' 설명.
5. **Tax Benefit Section (비과세)**:
   - 비과세 한도(월 150만원) 및 조건 설명.
6. **FAQ Section**:
   - 가입 조건, 중도 해지 관련, 비과세 상세 조건 등.
7. **Consultation Form (하단)**:
   - 이름, 연락처, 성별, 연령대 등 입력 폼.
   - 개인정보 수집 및 이용 동의.

## 4. 디자인 방향성 (Design Aesthetics)
- **신뢰감**: 전문적이고 신뢰감을 주는 블루/네이비 톤과 골드 포인트를 사용하여 프리미엄 느낌 강조.
- **가독성**: 폰트(Inter, Roboto, Outfit 등)를 사용하여 핵심 수치(8%, 연금액 등)를 시각적으로 부각.
- **반응형**: 모바일 기기에서 최적화된 레이아웃 제공.
- **동적 요소**: 스크롤 애니메이션, 호버 효과 등을 통해 사용자 참여 유도.

## 5. 기술 스택
- Core: HTML5, JavaScript (Vanilla JS)
- Styling: CSS (Modern layouts with Flexbox/Grid)
- Typography: Google Fonts (Pretendard 또는 Noto Sans KR 권장)
- Assets: AI 생성 이미지 및 고품질 라이브러리 사용.

## 6. 전환 목표 (Conversion Goal)
- 상담 신청 폼 제출 완료.
- 카카오톡 상담 버튼 클릭.
